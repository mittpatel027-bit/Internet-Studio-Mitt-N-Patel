About the Design

I designed this website with the intention of making it simple and sleek to let the images shine. I utilized blue and orange to give reference to the Chicago flag. Obviously the colors aren't an exact match, but I hoped the deeper and richer tones brought better contrast to the text and images. Lastly, I utilized the fonts "Inter" and "Krub" as my header and paragraph fonts. Inter feels very stylish yet charming. It pairs well with Krub as Krub is slightly more thin and quirky looking.


The structure of website is a simplified version of common travel/visiation websites. I also chose a scrolling format so I can have my images be slightly larger. I think if I would have done this assignment again, I would focus really hard on making sure every part of the website is alligned. I struggled a lot with the padding margin spacing of each element. However, I am still happy with my final product as I previously had no coding skills at all!
